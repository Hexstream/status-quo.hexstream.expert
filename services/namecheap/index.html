⬆ ⬈ ⚓ Namecheap » My config » DNS

I use Namecheap's free DNS service only for my trivial and not particularly active DNS zones.
Their DNS management interface is a bit clunky but it doesn't matter much when I have only 2 or 3 records that almost never need updates.

For other, more important and complex DNS zones, I use Route 53, which provides a world-class DNS service for only 0.50$ a month per zone.
(Too bad they don't yet provide secondary DNS...)
